Dalam algoritma Quick Sort, sebuah urutan dalam suatu array mempengaruhi running time algoritma tersebut. Dalam hal ini, kita melihat apabila suatu urutan array berawal dari bilangan terkecil, atau bilangan acak secara random bisa mempengaruhi bagaimana algoritma ini bekerja secara rekursif yang di mana pada kasus optimal bisa menjadi O (log n) atau kasus terburuk, yakni O(n^2). Karena optimasi pada algoritma ini belum tentu bisa menjadi solusi tercepat untuk suatu kasus.
